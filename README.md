# TaskFlow

A local-first task management app built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Kanban board with Backlog, In Progress, Review, and Done columns
- List view with the same task data
- Search plus status and priority filters
- Create, edit, delete, and update task status
- Browser persistence through `localStorage` at `taskflow.tasks.v1`
- First-run sample tasks

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

```bash
npm run lint
npm run build
npm run start
```
